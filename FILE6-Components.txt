REACT-COMPONENTS
    
    React components is a javascript function that you can sprinkle with markup.

    Lets deep dive into writing our first component.
    1. Export the component.
    2. Defining the function.
    3. Implement the markup.
    4. call the component.

    NOTE: Never define a component inside another component.

    1. Export default function functionName(){
        return(
            MARKUP
        );
    }

    2. You Can't write a component inside another component.

    3. The first alphabet of the component function name should be uppercase.
